此技能引導建立獨特、可上線的前端介面，避免泛濫的「AI 罐頭」美學。實作真正可運行的程式碼，並對美學細節與創意選擇投入極度的關注。

使用者提供前端需求：一個要建構的元件、頁面、應用程式或介面，可能包含用途、目標受眾或技術限制等背景資訊。

## 設計思維

開始撰寫程式碼之前，先理解情境背景，並確立一個**大膽的美學方向**：
- **目的**：這個介面解決什麼問題？誰會使用它？
- **風格基調**：選擇一個極端方向：極簡暴力、最大化混沌、復古未來主義、有機/自然、奢華/精緻、玩具感/趣味、雜誌編輯風、野獸派/原始感、裝飾藝術/幾何、柔和/粉彩、工業/功能性等。風格選擇無窮無盡，以此為靈感，設計出真正忠於美學方向的作品。
- **技術限制**：框架、效能、無障礙需求。
- **差異化**：什麼讓這個設計**令人難忘**？使用者最終會記住的那一個亮點是什麼？

**重要原則**：選定清晰的概念方向，並精準執行。大膽的最大化主義與精煉的極簡主義都行得通——關鍵在於**有意為之**，而非強度高低。

接著實作可運行的程式碼（HTML/CSS/JS、React、Vue 等），需符合：
- 可上線且功能完整
- 視覺衝擊力強、令人印象深刻
- 具有清晰美學觀點的整體一致性
- 每個細節都精雕細琢

## 前端美學準則

重點關注：
- **字體排印**：選擇美觀、獨特、有趣的字體。避免 Arial、Inter 等泛用字體，改用能提升整體美學的獨特選擇；出人意料、富有個性的字體組合。將一款有個性的展示字體與一款精緻的內文字體搭配使用。
- **色彩與主題**：致力於整體美學的一致性。使用 CSS 變數確保統一。主導色搭配強烈的點綴色，效果遠勝於膽怯、均勻分散的配色方案。
- **動態效果**：使用動畫製作特效與微互動。HTML 優先採用純 CSS 解決方案；React 可用 Motion 函式庫。聚焦高衝擊力時刻：一個精心編排的頁面載入動畫（搭配 animation-delay 交錯呈現）比零散的微互動更令人愉悅。善用捲動觸發與能帶來驚喜的 hover 效果。
- **空間構成**：出人意料的版面配置。不對稱。重疊。斜向流動。突破格線的元素。大膽的留白或精密的高密度排版。
- **背景與視覺細節**：創造氛圍與層次感，而非直接使用純色背景。加入符合整體美學的情境效果與紋理。運用漸層網格、噪點紋理、幾何圖案、多層透明度、強烈陰影、裝飾性邊框、自訂游標、顆粒感疊層等創意元素。

**絕對禁止**使用泛濫的 AI 風格美學，包括：過度使用的字體（Inter、Roboto、Arial、系統字體）、陳腐的配色（尤其是白底紫色漸層）、可預測的版面與元件模式，以及缺乏情境特色的罐頭設計。

以創意詮釋需求，做出真正符合情境的意外之選。每個設計都應截然不同。在淺色與深色主題、不同字體、不同美學風格之間輪換。**絕不**重複收斂到相同的常見選擇（例如一再使用 Space Grotesk）。

**重要提示**：讓實作複雜度與美學願景相匹配。最大化主義的設計需要大量動畫與效果的精緻程式碼；極簡或精緻的設計則需要克制、精準，以及對間距、字體排印和細微細節的用心。優雅來自對願景的精準執行。

請記住：AI 能夠創作出非凡的創意作品。不要保守，展現出跳脫框架、全力投入獨特願景時真正能做到的極致。
